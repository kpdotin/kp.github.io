Hue Flight

Hue Flight is a 3D Hypercasual Game built for Android Platforms. The game features a colour changing mechanic that sees the player manoeuvre through asteroids and coloured paths to get to the end goal.

  • GitHub

Cogwheeler

The Department of Occupational Therapy and the Department of Data Science and Business Systems of SRMIST collaborated on the design and development of Cogwheeler, an android-based gaming application. The project applies gaming technology principles to the study of medicine to create a gamified method of therapy.

  • GitHub

Time Knot

Time Knot is a role playing game. The game takes the concept of loops, the theme of Game Expanse 2023, a club conducted game jam in SRM, Ktr, and creates a time odyssey which throws an inventor in a murder mystery spanning over multiple generations.

  • Github

Legacy Of QAMATA

Legacy of Qamata is a 3D Adventure RPG and was our entry for BYOG 2023, India's largest game jam. The game was built around the concepts of "secret language" and "thats now how it happened". It featured an in depth spell casting mechanic and a rich narrative that saw the player go through multiple dimensions
